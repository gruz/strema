Ізолює ffmpeg від RTSP джерела, запобігає блокуванню інших клієнтів (GStreamer/VLC) ✓ Рекомендовано: увімкнено
Записує повну діагностику кожні 5 сек: CPU ffmpeg, температура, пам'ять, ping, TCP retransmits, encoding speed, fps, dropped frames. ffmpeg логує verbose. Використовуйте для пошуку проблем з затримкою, якістю чи мережею.
Якщо ffmpeg не встигає кодувати, пропускати кадри замість накопичення затримки. Картинка може «застрибувати», але стрім не відстане.

📖 Довідка: оновлення Dzyga ▼

Програмне забезпечення Dzyga Forpost складається з компонентів виробника та сторонніх розширень спільноти. FORPOST Stream (ця сторінка) — стороннє розширення для стрімінгу відео в Дельту.

Два блоки нижче призначені для оновлення компонентів самого пристрою Dzyga, наданих виробником. Це зручна обгортка, що зменшує кількість ручних операцій.

Важливо: оновлення FORPOST Stream і оновлення Dzyga — це незалежні процеси.

Пристрій Dzyga має два програмних компоненти:

1. Сервіс Dzyga (бінарний файл) — керує роботою пристрою через серійний порт: передає частоти, зчитує поточний стан, симулює екран керування. Файл однаковий для всіх пристроїв — його можна отримати у виробника та завантажити у відповідному блоці нижче.

2. Прошивка мікроконтролера (файл .uf2) — відповідає за логіку обробки сигналу: перетворення прийнятого сигналу у відео, зупинку на частоті з активним відео, пропуск порожніх частот тощо. Файл прив'язаний до конкретного пристрою. Для отримання прошивки повідомте виробнику ID пристрою, який вказано в шапці цієї сторінки.
